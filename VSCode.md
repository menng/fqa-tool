### 常用设置

- Editor: Mouse Wheel Zoom 打勾，可以按着ctrl放大字体
- File -> Auto Save 自动保存
- File -> renderWhitespace 展示空格

### 快捷键

- Alt + Shift + 左键 选择列
- Ctrl + b 切换侧边栏
- Ctrl + p 查询文件
- Ctrl + Shift + p 命令面板
